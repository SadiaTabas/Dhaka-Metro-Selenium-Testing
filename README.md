# Dhaka-Metro-Selenium-Testing
Automated UI testing of a Metro Rail Ticket Management System using Selenium IDE.
 

This repository contains automated UI testing for the Dhaka Metro Rail Ticket Management System using Selenium IDE.

## Project Overview

The project demonstrates automated functional testing of a metro rail ticketing web application. Multiple user roles were tested including Passenger, Admin, and Employee.

## Tools Used

- Selenium IDE
- Firefox
- Web Browser
- Manual Testing
- Functional Testing

 

## Test Scenarios

### Passenger Module

- Login
- Logout

- Ticket Payment

- Payment History
- Ticket History

### Admin Module

- Dashboard
- Add Route
- Delete Route
- Add Staff
- Delete Staff
- Update Profile

### Employee Module

- Dashboard
- Update Profile

## How to Run

1. Install Firefox Browser
2. Install Selenium IDE Extension
3. Open Selenium IDE
4. Select **Open Existing Project**
5. Open `Dhaka Metro.side`
6. Click **Run All Tests**
